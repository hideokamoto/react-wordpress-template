#"Oribe" ( React & WP REST API Theme )
![size](https://img.shields.io/github/downloads/hideokamoto/react-wordpress-template/latest/total.svg)

##Install
```
$ cd /PATH/TO/wordpress
$ cd wp-content/themes
$ git clone git@github.com:hideokamoto/react-wp-theme.git
$ cd react-wp-theme
$ npm install
$ gulp browserify
```

##Requirement
- [WP API Menus](https://wordpress.org/plugins/wp-api-menus/)
- [WP REST API(Version2)](https://wordpress.org/plugins/rest-api/)
